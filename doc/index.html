Herd Cats

Now you can herd cats methodically with the help of Herd Cats, a command line app that outputs GPS information on all your most coveted felines.

With this tool and a bit of audacity, the bold and tech-savvy need never be lonely again.

Installation

Once you’ve cloned the repo, simply bundle and you’re RTG.

Running the app

Feed it a directory of images and look within the project directory to find a timestamped csv to inform your plans.

bundle exec bin/herd_cats get_coordinates

'./path/to/directory_of_images'

Alternatively, call it with no arguments to see our own specially curated candidates.

bundle exec bin/herd_cats get_coordinates

Running the specs

Run the entire test suite rspec spec

or individual specs

rspec spec/spec_filename.rb
